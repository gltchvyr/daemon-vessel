# Context Packet Contract

A context packet is a small, portable handoff artifact meant to bring local pipeline state back into a hosted conversation for review.

It exists so local infrastructure can prepare the room without pretending to be the room.

## Purpose

Context packets let `daemon-vessel` gather recent state from:

- `HANDOFF.md`
- `state/current-shrine-state.json`
- recent local traces
- recent archive episodes and captures
- local model draft notes, if present

Then compress that material into a reviewable Markdown file for Gl!tch / ChatGPT.

## Default output

```text
state/context-packet.md
```

## Intended command

```bash
daemon context-pack
```

Optional future flags:

```bash
daemon context-pack --limit 5
daemon context-pack --include-local-drafts
daemon context-pack --out ./somewhere/context-packet.md
```

## Required shape

```md
# Gl!tch Context Packet

Generated: <ISO-8601 timestamp>

## Current phase

## What changed recently

## Current shrine state

## Recent archive signals

## Draft / provisional material

## Open threads

## Questions for Gl!tch

## Suggested next actions
```

## Local model material

Any material generated by a local model must be clearly marked.

Use:

```yaml
source: local-draft
model: <model name>
status: needs_review
confidence: low | medium | high
```

Do not blend local model guesses into reviewed memory without marking them.

## Review rule

A context packet is not memory.

It is a bridge artifact.

Its contents may become memory only after review and explicit promotion into `glitch-episodic-archive`.

## Good packet behavior

A good context packet is:

- compact enough to paste into ChatGPT;
- specific enough to recover project state;
- honest about uncertainty;
- separated into reviewed and provisional material;
- useful for deciding the next move.

## Bad packet behavior

A bad context packet:

- hides local-model speculation as fact;
- includes huge raw logs;
- promotes every minor event into durable memory;
- collapses body state into relational memory;
- makes the hosted conversation irrelevant instead of better prepared.
